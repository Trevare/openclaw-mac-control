# PRD — OpenClaw Mac Control (MVP v1)

## Цель
Сократить время подключения к OpenClaw и восстановления множества OpenAI-аккаунтов.

## Пользовательские сценарии
1. Ввожу IP/логин/пароль сервера → проверяю подключение.
2. Добавляю OpenAI-профили (alias + заметка + токен/ключ в Keychain).
3. Запускаю «Restore Wizard» и прохожу аккаунты по шагам.
4. Получаю статус: какие профили успешно восстановлены.

## Нефункциональные требования
- Хранение секретов только в macOS Keychain
- Логи без секретов
- Экспорт только шифрованный
